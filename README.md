# provider-harbor

`provider-harbor` is a [Crossplane](https://crossplane.io/) Provider for Harbor.

Work still in progress ...


## Developing

Run against a Kubernetes cluster:

```console
make run
```

Build, push, and install:

```console
make all
```

Build image:

```console
make image
```

Push image:

```console
make push
```

Build binary:

```console
make build
```
